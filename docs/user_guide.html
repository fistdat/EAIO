Energy AI Optimizer - User Guide

Introduction

Welcome to the Energy AI Optimizer system user guide. This guide provides comprehensive information on how to use the platform based on your role within the organization. The system is designed to help you monitor, analyze, and optimize energy consumption in your buildings through an intelligent multi-agent AI system.

The Energy AI Optimizer serves three primary user roles, each with specific features and interfaces tailored to their needs:

  • Facility Managers - Day-to-day operational management and immediate actions
  • Energy Analysts - Detailed technical analysis and optimization strategies
  • Executives - Strategic overview and financial/sustainability metrics

System Overview

The Energy AI Optimizer utilizes a multi-agent AI architecture to provide:

  • Real-time energy consumption monitoring and anomaly detection
  • Weather-normalized energy usage analysis
  • AI-powered recommendations for energy savings
  • Forecasting of future energy consumption patterns
  • Role-based dashboards and reports
  • Natural language interface for energy-related queries
Note: The system processes data from the Building Data Genome 2 (BDG2) dataset by default, but can be configured to work with your building's specific energy management system.

Role-Specific Guides

For Facility Managers

As a facility manager, you'll primarily use the system for day-to-day operational oversight and quick response to energy-related issues.

Key Features

  • Operational Dashboard - Real-time view of building systems with alert indicators
  • Anomaly Alerts - Immediate notification of unusual energy consumption patterns
  • Quick Actions - Recommended immediate steps to address issues
  • System Health - Status overview of HVAC, lighting, and other building systems
  • Daily Reports - Summary of the previous day's performance and issues

Common Tasks

Task Navigation
Check current building status Dashboard → Overview
Review today's alerts Dashboard → Alerts
View recommended actions Dashboard → Recommendations
Schedule system adjustments Operations → Schedule
Generate daily report Reports → Daily Summary

For Energy Analysts

Energy analysts will use the platform for detailed technical analysis and developing optimization strategies.

Key Features

  • Advanced Analytics - In-depth analysis tools with customizable parameters
  • Energy Modeling - Create and test energy consumption models
  • Benchmarking - Compare performance against similar buildings and standards
  • Technical Recommendations - Detailed technical optimization suggestions
  • Custom Reports - Generate specialized reports with specific metrics

Common Tasks

Task Navigation
Perform trend analysis Analytics → Trends
Create energy model Analytics → Modeling → New Model
View weather correlation Analytics → Weather Impact
Run efficiency simulations Optimization → Simulate
Generate technical reports Reports → Technical Analysis

For Executives

Executives will focus on high-level overview, financial metrics, and strategic decision-making information.

Key Features

  • Executive Dashboard - High-level KPIs and financial metrics
  • Cost Analysis - Energy cost trends and savings opportunities
  • Sustainability Metrics - Carbon footprint and environmental impact
  • Strategic Recommendations - Long-term improvement opportunities with ROI
  • Portfolio View - Multi-building comparison and portfolio performance

Common Tasks

Task Navigation
View financial overview Executive → Financial Summary
Check sustainability goals Executive → Sustainability
Review strategic recommendations Executive → Strategic Opportunities
Compare buildings in portfolio Executive → Portfolio View
Generate executive report Reports → Executive Summary

Using the Natural Language Interface

All users can access the natural language interface to query the system using plain English questions. This feature allows you to quickly get information without navigating through menus.

Example Queries

  • "Show me yesterday's energy consumption compared to last week"
  • "Which building system is using the most energy right now?"
  • "What are the top 3 recommendations to reduce energy costs this month?"
  • "How has the recent weather affected our energy usage?"
  • "Generate a report on HVAC efficiency for the past 30 days"

To use this feature, click on the chat icon in the top right corner of any page and type your question.
